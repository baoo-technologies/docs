## Time range

A dropdown in the top‑right corner lets you adjust the time range:

<div className="my-4">
  <img
    src="/images/usage-filter.png"
    alt="Time‑range dropdown"
    className="rounded-lg shadow"
  />
</div>

| Option                                     | What it shows                                          |
| ------------------------------------------ | ------------------------------------------------------ |
| **Current billing period**                 | Default view – tracks credits against your plan limit. |
| **Yesterday / Last 7 days / Last 30 days** | Quick looks at recent traffic.                         |
| **Custom**                                 | Pick any start & end date for ad‑hoc analysis.         |
