<Note>
  Homie exposes every chat capability through a simple JSON/HTTPS API. To get
  started you only need a **server‑side API key** and the **chatbotId** of the
  assistant you created in the dashboard.
</Note>

## 3‑Step Flow

| Step               | Endpoint                               | Purpose                                                                                 |
| ------------------ | -------------------------------------- | --------------------------------------------------------------------------------------- |
| 1 • Create chat    | **POST** `/v1/chats`                   | Spin up a new conversation and receive its `chatId`. Pass your `chatbotId` in the body. |
| 2 • Add messages   | **POST** `/v1/chats/{chatId}/messages` | Send user text, receive Homie’s answer (plus product recommendations, flags, etc.).     |
| 3 • Listen / fetch | **GET** `/v1/chats/{chatId}/events`    | (Optional) Stream events in real time or poll your preferred endpoint.                  |

### 1. Create a chat

```bash
curl -X POST https://api.homie.ai/v1/chats \
  -H "Authorization: Bearer $HOMIE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "chatbotId": "cbt_1234567890",
    "userId": "user_42"
  }'
```

**Response**

```jsonc
{
  "id": "chat_abc123",
  "chatbotId": "cbt_1234567890",
  "userId": "user_42",
  "organizationId": "org_99",
  "createdAt": "2024-06-12T07:24:01Z",
}
```

Save the `id` — you’ll need it for every follow‑up call.

### 2. Send the first message

```bash
curl -X POST https://api.homie.ai/v1/chats/chat_abc123/messages \
  -H "Authorization: Bearer $HOMIE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Does the blue hoodie come in size L?",
    "currentUrl": "https://store.com/products/blue-hoodie"
  }'
```

Homie replies with structured data:

```jsonc
{
  "id": "msg_987",
  "chatId": "chat_abc123",
  "type": "answer",
  "content": "Yes! Size L is in stock…",
  "products": [
    {
      "gtin": "1234567890123",
      "brand": "Homie",
      "designation": "Blue Hoodie",
      "uri": "https://store.com/products/blue-hoodie",
      "image": "https://store.com/images/blue-hoodie.png",
      "price": 59.99,
      "currency": "EUR",
      "availability": { "status": "in_stock", "quantity": 12 },
    },
  ],
  "createdAt": "2024-06-12T07:24:05Z",
}
```
